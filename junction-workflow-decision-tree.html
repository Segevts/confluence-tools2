Roundabout Special Rules

Same propagation rules apply, with additional considerations

Entry Speed Rule

Each speed entering the roundabout becomes the new speed inside the RA at that point

Speed Changes Inside RA

Keep each entry speed until you encounter a new speed from another entry point

Exit Propagation

Same propagation rules apply when exiting the roundabout

Complex RAs (3+ speeds)

Escalate to Team Lead for roundabouts with 3 or more different entry speeds

Final Speed Assignment

Apply Determined Speed
Set the Legal Speed value in the DP
Select appropriate Confidence value
based on the determination method
Update all affected DPs
Change speed at the point where new road section starts
(not inside the turn/junction)
